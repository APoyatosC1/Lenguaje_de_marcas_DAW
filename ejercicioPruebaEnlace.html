HTML

HTML, siglas de HyperText Makeup Language, hace referencias al lenguaje de amrcado para la elaboración de páginas web. Es un estandar que sirve de referencia para la elaboración de páginas web en sus diferentes versiones, define una estructura básoca y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imagenes, entre otros. Es un estándar a cargo de la W3C, organizacion dedicada a la estandarizacion de casi todas las tecnologias ligadas a la web, sobre todo en lo refenrente asu escritura e interpretacion. Es el lenguaje con el que se definen las paginas web
